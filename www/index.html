Secure & Anonymous

Although posts are made by pseudonymous IDs, the IDs are in no way tied to anything (e.g. e-mails, phone numbers). This simplistic architecture leads to IDs being anonymous and also impossible to fake or spoof. This offers full anonymity and at the same time security.

Uncensored & Trustless

Since there are no accounts, and only mathematically derived IDs (which can be performed anywhere) it is impossible to delete a user. The only thing saved on our servers are IDs, which are public information anyway. Because of this, FUS is an extremely lightweight and a trust minimized system.

Transparent

FUS client and server side code is open source and thereby fully transparent. The Database is transparent since there is no "secret" data and all data is displayed publicly. Any and all data being public, adds to the security.
